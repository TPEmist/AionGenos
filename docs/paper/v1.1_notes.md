## FINDING (holds regardless of L2's fate) — memory needs a success floor

L2 collect gave teacher SR = 1/100; the recap buffer is 1 desirable +
99 undesirable. Stated as a finding, not a mishap:

**Memory can only supply *conditional* knowledge if there exist success
experiences to condition on. A retrieval buffer built from a
near-failing teacher retrieves "others also failed here" — it cannot
supply the situation→correct-correction map that is the whole value of
memory (§4.3).** This is the cold-start / minimum-initial-competence
threshold (cf. SimpleVLA-RL's RL cold-start) reappearing on the *memory*
side: below a task-competence floor, externalising memory has nothing to
externalise — the same "minimum viable competence" law seen from the
memory face rather than the RL face.

One paragraph in Discussion (future work / scope): the marginal/
conditional criterion presupposes the teacher clears a task-competence
floor; L2 dual-push at 1% SR sits below it, which is itself evidence for
the floor and a boundary condition on "distil the competence,
externalise the memory" — you cannot externalise a competence the
teacher does not have.
